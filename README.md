# Smart Expanse Tracker

A Full-Stack MERN (MongoDB, Express.js, React, Node.js) application designed to help users manage and track personal finances. The dashboard provides real-time financial insights through dynamic calculations, interactive charts, and a responsive user interface.

---

## Overview

Finance Tracker Dashboard enables users to record income and expenses, monitor financial activity, and visualize spending patterns through interactive charts and real-time calculations.

---

## Features

* Add and manage income and expense transactions
* Real-time balance, income, and expense calculations
* Interactive financial visualization using Chart.js
* Category-based expense tracking
* Dark/Light mode toggle
* Responsive and user-friendly interface
* RESTful API with MongoDB integration

---

## Tech Stack

### Frontend

* React.js
* Vite
* Axios
* Chart.js

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Installation & Run

```bash
git clone https://github.com/YOUR_USERNAME/finance-tracker.git
cd finance-tracker

npm install
npm run dev
```

Open:

```text
Frontend: http://localhost:5173
Backend:  http://localhost:5000
```
